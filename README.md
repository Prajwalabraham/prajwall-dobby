
# Dobby-assesment

An assesment asked by a company to understand the skillset.




## Installation

Install Dobby-assesment with npm

Client:
```bash
  git clone https://github.com/Prajwalabraham/Dobby-assesment.git
  cd Dobby-assesment
  npm install
  npm start
```
Server:

```
    cd Backend
    npm start
```


## Features

- Login/Signup
- Image Uploads
- Logout 
- Image preview
